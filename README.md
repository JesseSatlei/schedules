## Description

Esse projeto foi criado utilizando o [Nest](https://github.com/nestjs/nest) framework TypeScript.
De meia em meia hora será enviado para o Logger uma consulta com os horarios disponíveis.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

